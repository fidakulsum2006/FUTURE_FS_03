# Fidu's Kitchen Restaurant Website

Fidu's Kitchen is a complete premium restaurant website project built with HTML, CSS, and JavaScript. The design is crafted for a five-star fine dining brand and uses a luxury white, green, and gold visual identity with rich animations, strong call-to-action sections, menu storytelling, reviews, and a reservation form.

## Project Files

- `index.html` - Main website structure and content
- `styles.css` - Premium visual design, layout, and animations
- `script.js` - Interactivity, menu filters, testimonial slider, counters, and form feedback
- `PITCH.md` - Short business pitch for the project submission

## Features

- Premium hero section with layered animated visuals
- Elegant white, emerald, gold, and champagne color system
- Signature menu section with interactive category filters
- Scroll reveal animations and animated counters
- Testimonial carousel
- Professional reservation form
- Fully responsive layout for desktop, tablet, and mobile
- Self-contained static website that can be opened directly in a browser

## How to Run

1. Open the project folder.
2. Double-click `index.html`.
3. Or run a local server if you want:

```powershell
python -m http.server 8000
```

Then open `http://localhost:8000`.
